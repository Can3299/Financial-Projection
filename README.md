# ProCalc — Financial Projection

An interactive financial projection calculator that visualizes investment growth over time using compound or simple interest.

## Features

- **Compound / Simple Interest** — Switch between calculation methods
- **Interactive Chart** — Growth timeline rendered with Chart.js
- **Dark Mode** — Toggle or auto-detects system preference
- **Responsive** — Tailwind CSS layout works on all screen sizes

## Usage

Open `src/index.html` in any browser. Enter principal amount, interest rate, duration, select a method, and click *Analyze & Update Chart*.
